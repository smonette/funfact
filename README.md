# FUN FACT!

Fun Fact is a simple Node.js app where you can post a fun fact that you know. It's built on a Postgres database. The Sass is compiled by a Grunt task, and will autoupdate when you run `grunt watch`

## Included Packages/Dependencies
* body-parser
* ejs
* express
* grunt
* nodemon
* pg
* sequelize
* grunt
* grunt-sass
* grunt-contrib-watch
